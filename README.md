## 🔢 Digit Frequency Counter 🚀


## 📌 Overview


Counts how many times each digit appears in a number entered by the user.
Built using pure mathematical logic (no string conversion).


🧠 Logic Used


🔁 While Loop


 🏹Modulus (%) to get last digit


 🏹Division with Math.floor() to remove last digit


📦 Object to store digit counts


## ⚙️ How It Works


1️⃣ Take number input


2️⃣ Extract last digit using % 10


3️⃣ Store & increment digit count


4️⃣ Remove last digit using division


5️⃣ Repeat until number becomes 0


## 📊 Example


Input: 1223


Output:


3 → 1 time


2 → 2 times


1 → 1 time


## 🎯 Purpose


Improves logical thinking, number manipulation, and JavaScript fundamentals 💪🔥
